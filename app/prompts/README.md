# Prompt Templates

These prompts are part of the AI CloudOps Incident Assistant control plane.

## Purpose
They make AI behavior predictable, auditable, and safer.

## Change control
- Review prompt changes before deployment.
- Test with simulated incidents.
- Keep safety rules explicit.
- Store prompts in Git.
